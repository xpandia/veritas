## Slide 5: HOW IT WORKS

**File. Deliberate. Resolve.**

### 1. FILE
A party submits the dispute on-chain: the contract, the evidence, the claim. Immutable. Timestamped. Transparent.

### 2. DELIBERATE
GenLayer's intelligent contract triggers arbitration. Multiple AI validators *independently* analyze the case — reading the contract, weighing evidence against applicable law, reasoning through each claim. Then they converge through GenLayer's consensus mechanism. No single point of failure. No bias. No backroom deals.

### 3. RESOLVE
A binding verdict is issued on-chain. Smart contract escrow releases funds to the prevailing party. Automatically. Instantly. Done.

No lawyers required. No hearings scheduled. No appeals that drag on for years.

## Slide 6: THE TECHNOLOGY

**Why GenLayer Changes Everything**

Traditional blockchains execute deterministic code. `if x > y, send funds.` They can't *reason*.

GenLayer is the first blockchain where smart contracts call LLMs, access external data, and reach consensus on *subjective* outputs. That's not an incremental improvement. That's a new category.

**Veritas's Intelligent Contract:**
- Written in Python (GenLayer's native contract language)
- Calls LLM validators to analyze disputes with full legal reasoning
- Uses GenLayer's optimistic rollup consensus — validators independently reason, then converge
- Executes binding outcomes through on-chain escrow

**This is the application GenLayer was built for.** Arbitration is inherently subjective. It requires reading, reasoning, and judgment. GenLayer is the only chain where that's possible natively.
